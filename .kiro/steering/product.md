# Product Overview

**Tokenomics Lab** (formerly Token Guard Pro) is a multi-chain cryptocurrency token risk analysis platform that provides AI-powered security assessments for tokens across Ethereum, BSC, Solana, Polygon, Arbitrum, and other blockchain networks.

## Core Value Proposition

Provides investors and developers with transparent, data-driven insights into token security and risk factors through:
- 10-factor risk algorithm with chain-adaptive scoring
- AI-powered meme token detection using Groq AI (Llama 3.3 70B)
- Real-time market data from multiple APIs (Mobula, Moralis, GoPlus, Helius)
- Historical analytics and behavioral pattern analysis
- Personal watchlist management with Firebase integration

## User Tiers

- **FREE**: 20 scans/day, basic risk analysis, 5 watchlist tokens, 2 charts
- **PREMIUM**: Unlimited scans, AI insights, advanced analytics, unlimited watchlist, 4 charts, alerts
- **ADMIN**: Full platform management, user role control, notification preferences

## Key Features

- Multi-chain token analysis (EVM + Solana + Cardano)
- Stablecoin detection with automatic low-risk override
- Critical flag detection (honeypot, mintable, freeze authority)
- Glassmorphism UI with monotone design aesthetic
- CoinMarketCap integration for token search
- Email notifications for tier changes
- GDPR-compliant privacy settings
