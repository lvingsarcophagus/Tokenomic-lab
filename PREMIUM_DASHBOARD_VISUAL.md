## API Integration Points

### 1. Watchlist API
```typescript
GET /api/user/watchlist
Response: {
  tokens: WatchlistToken[]
}

POST /api/user/watchlist
Body: { address, chain, chainId }
Response: { success: true, token: WatchlistToken }

DELETE /api/user/watchlist/{address}
Response: { success: true }
```

### 2. Portfolio Stats API
```typescript
GET /api/user/portfolio-stats
Response: {
  totalTokens: 12,
  averageRiskScore: 34,
  criticalTokens: 1,
  totalScans: 145,
  behavioralInsights: 8
}
```

### 3. Alerts API
```typescript
GET /api/user/alerts
Response: {
  alerts: Alert[]
}

PATCH /api/user/alerts/{alertId}
Body: { read: true }
Response: { success: true }
```

### 4. Historical Data API
```typescript
GET /api/user/analysis-history?tokenAddress=xxx&days=30
Response: {
  history: [{
    timestamp: number,
    riskScore: number,
    price: number,
    holderCount: number
  }]
}
```
